# Dog Gallery

App de galería de imágenes que consume una API simple a través de la librería Retofit.

## Características

* Retrofit
* Dagger Hilt
* MVVM
* Clean Arquitecture
* Jetpack Compose

# Información

Esta app consume la API de [Dog Images](https://dog.ceo/api/). Accede a la misma para mejor
comprensión y uso.
